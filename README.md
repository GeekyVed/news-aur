# News Fetcher

A professional, terminal-based news reader for Computer Science and AI.

## Features

- **Curated Content**: Fetches only relevant CS, AI, and Tech news.
- **Deterministic**: Shows the latest news sorted by date, not random picks.
- **Clean Interface**: Colorized output with clickable links.
- **Fast**: Caches results to minimize network requests (optional).

## Usage

```bash
news
```

## Installation

### Arch Linux (PKGBUILD)

```bash
makepkg -si
```

### Manual

```bash
chmod +x news.py
./news.py
```

## Requirements

- Python 3
